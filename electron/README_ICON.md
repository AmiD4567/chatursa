# Инструкция по созданию иконки

Для работы приложения нужна иконка в формате `.ico`.

## Вариант 1: Онлайн конвертер
1. Создайте изображение 256x256 пикселей (PNG)
2. Конвертируйте в ICO на сайте: https://convertio.co/png-ico/
3. Сохраните как `electron/icon.ico`

## Вариант 2: Использовать готовую
Поместите любой файл `icon.ico` в папку `electron/`

## Вариант 3: Создать через PowerShell (Windows)
```powershell
# Установите ImageMagick или используйте онлайн конвертер
```

## Вариант 4: Временное решение
Без иконки приложение будет использовать стандартную иконку Electron.
