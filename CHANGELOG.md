# 📋 История изменений

> Автоматически сгенерировано: 28 марта 2026 г.

---

**Примечание:** Этот файл будет автоматически обновляться при каждом коммите через `smart-commit.bat` или `commit-changes.bat`.

## Как использовать

1. **Внесите изменения** в код проекта
2. **Запустите** `smart-commit.bat` (рекомендуется) или `commit-changes.bat`
3. **Введите описание** изменений
4. **Готово!** CHANGELOG.md обновится автоматически

## Типы изменений

| Тип | Описание |
|-----|----------|
| ✨ Feature | Новая функция |
| 🐛 Fix | Исправление ошибки |
| 🔒 Security | Изменения безопасности |
| ⚡ Performance | Улучшение производительности |
| 📝 Docs | Обновление документации |
| 🎨 Style | Изменения стиля/дизайна |
| ♻️ Refactor | Рефакторинг кода |
| 🧪 Tests | Добавление тестов |
