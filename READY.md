## ⚙️ Настройки пользователя

Настройки оформления сохраняются в:
```
%APPDATA%\Chat App\userUiSettings_{userId}
```

База данных:
```
%LOCALAPPDATA%\Chat App\backend\chat.db
```

## 🐛 Решение проблем

### Приложение не запускается
1. Проверьте что Windows 10/11 x64
2. Запустите от имени администратора
3. Проверьте антивирус

### Нет уведомлений
1. Проверьте настройки уведомлений Windows
2. В приложении: Настройки → Уведомления

### Ошибка при установке MSI
1. Закройте приложение перед переустановкой
2. Удалите старую версию через Панель управления
3. Запустите установщик заново
