# 🎵 Как добавить свой звук уведомлений

## 📁 Шаг 1: Подготовьте звуковой файл

Ваш звуковой файл должен быть в формате:
- **MP3** (рекомендуется)
- WAV
- OGG

## 📋 Шаг 2: Скопируйте файл в папку проекта

Скопируйте ваш звуковой файл в папку:

```
c:\Users\user\Desktop\Чат\chat-app\frontend\public\sounds\
```

И переименуйте его в:

```
notification.mp3
```

**ИЛИ**

Если вы хотите использовать другое имя файла, измените строку в `frontend/src/App.js`:

```javascript
const audio = new Audio('/sounds/ВАШ_ФАЙЛ.mp3');
```

## 🔄 Шаг 3: Пересоберите frontend

После добавления файла нужно пересобрать frontend:

```bash
cd c:\Users\user\Desktop\Чат\chat-app\frontend
npm run build
```

## ✅ Готово!

Теперь при получении новых сообщений и дней рождениях будет проигрываться ваш звук!

---

## 📝 Примечания

- Файл должен находиться именно в `frontend/public/sounds/`
- После изменения нужно обязательно сделать `npm run build`
- Звук работает для:
  - ✅ Новых сообщений
  - ✅ Уведомлений о днях рождениях
