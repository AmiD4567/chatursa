**Файл:** `dist/Chat App 1.0.0.msi`  
**Размер:** 127.6 МБ

## 🚀 Установка

1. Закройте все экземпляры приложения (если открыты)
2. Запустите `dist/Chat App 1.0.0.msi`
3. Следуйте инструкциям установщика
4. Приложение запустится автоматически

## 🔍 Как проверить что всё работает:

### Первый запуск:
1. Откройте приложение из меню Пуск или с рабочего стола
2. Должно открыться окно чата
3. В трее (возле часов) появится иконка

### Повторный запуск:
1. Попробуйте открыть приложение ещё раз
2. Должно появиться сообщение: "Приложение уже запущено!"
3. Существующее окно развернётся

### Сворачивание:
1. Закройте окно (крестик)
2. Приложение свернётся в трей
3. Двойной клик по иконке в трее восстановит окно

## 📝 Логи

Приложение пишет логи в консоль. Для просмотра:
1. Запустите приложение
2. Проверьте консоль (если есть)
3. Или посмотрите логи в `%APPDATA%\Chat App\logs\`

## 🐛 Решение проблем

### Приложение не открывается
1. Проверьте трей (возле часов) - возможно оно уже работает
2. Закройте через трей (правый клик → Выйти)
3. Запустите заново

### Видны процессы но нет окна
1. Проверьте трей
2. Двойной клик по иконке в трее
3. Или закройте процессы в диспетчере и запустите снова

### Ошибка при запуске
1. Удалите приложение через Панель управления
2. Переустановите из MSI
3. Запустите от имени администратора
