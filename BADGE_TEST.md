# Проверка работы красной точки

## ВАЖНО: Полностью закройте приложение перед установкой новой версии!

1. Закройте приложение из трея (правый клик -> Выйти)
2. Убедитесь что процесс electron.exe закрыт в Диспетчере задач
3. Установите новую версию
4. Запустите приложение

## Проверка в DevTools

После запуска приложения:

1. Откройте DevTools (F12)
2. Перейдите на вкладку Console
3. Выполните команду:

```javascript
window.electronAPI.setUnreadCount(3)
```

4. Проверьте логи в файле: `%APPDATA%\chat-app\logs\chat-app.log`

Если в логах появятся записи про `set-unread-count` и `createBadgeIcon`, но точки нет - значит проблема в отрисовке.

Если записей нет - значит `window.electronAPI.setUnreadCount` не вызывается.
