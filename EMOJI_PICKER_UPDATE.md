4. **Смайл добавится** - в поле ввода сообщения

## 🎨 Настройки EmojiPicker

В `App.js` можно изменить параметры:

```jsx
<EmojiPicker 
  onEmojiClick={handleAddEmoji}
  theme="dark"              // 'dark' | 'light' | 'auto'
  emojiStyle="native"       // 'native' | 'apple' | 'facebook' | 'twitter' | 'google'
  searchDisabled={false}    // отключить поиск
  skinTonesDisabled={false} // отключить выбор тона кожи
  reactionsDefaultOpen={false}
/>
```

## 🔧 Дополнительные настройки

### Изменить позицию:
В `index.css` найдите `.emoji-picker-container` и измените:
```css
.emoji-picker-container {
  position: absolute;
  bottom: 100%;      /* позиция по вертикали */
  left: 50%;         /* позиция по горизонтали */
  transform: translateX(-50%);
}
```

### Изменить тему:
В `App.js` измените `theme="dark"` на `theme="light"`

### Изменить стиль эмодзи:
В `App.js` измените `emojiStyle="native"` на:
- `"apple"` - стиль Apple
- `"facebook"` - стиль Facebook
- `"twitter"` - стиль Twitter
- `"google"` - стиль Google

## 🐛 Возможные проблемы

### EmojiPicker не появляется:
- Проверьте, что `showEmojiPicker` переключается
- Проверьте z-index в CSS

### Смайлы не добавляются:
- Проверьте `handleAddEmoji()` функцию
- Убедитесь, что `messageInputRef` корректно работает

### Неправильная тема:
- Проверьте `theme` prop в `<EmojiPicker />`
- Обновите стили в `index.css`
